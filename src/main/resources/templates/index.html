# Name End Point URL Comments
1 Search Restaurants(locu) http://bluedino.ng.bluemix.net/restaurant/search Any or none of the following search parameters can be added:
  • name
  • locality
  • cuisine
  • region
  • postal_code
  • country
  • radius
  • street_address
  • open_at
  • has_menu
  • bluedino.ng.bluemix.net/restaurant/search?locality=Pittsburgh&name=mintt
2 Search Menu List(locu) http://bluedino.ng.bluemix.net/restaurant/search/{restaurantid}/menu restaurantid can be found in json response field name "id"
  • http://bluedino.ng.bluemix.net/restaurant/search/db4fd557bde067351208/menu
3 Fetch all Restaurants(dino) http://bluedino.ng.bluemix.net/restaurant Fetches a list of json restaurant objects
  • http://bluedino.ng.bluemix.net/restaurant
4 Fetch specific Restaurant(dino) http://bluedino.ng.bluemix.net/restaurant/{restaurantid} Fetches json restaurant object
  • http://bluedino.ng.bluemix.net/restaurant/6ba336aadbcbe1e62f0b
5 Fetch Menus for Restaurant(dino) http://bluedino.ng.bluemix.net/restaurant/{restaurantid}/menu Fetches a list of json menu objects
  • http://bluedino.ng.bluemix.net/restaurant/6ba336aadbcbe1e62f0b/menu
